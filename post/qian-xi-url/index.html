浅析 URL

2020-08-04

URL 包含哪几部分，每部分分别有什么作用
URL包括协议+域名或IP+端口号+路径+查询字符串+锚点
• 协议就是网站用的协议，一般是http或https
• IP申请的访问地址，可以通过域名访问，也是IP的别称
• 不同的协议拥有不同端口号
• http是80
• https是443
• 路径是指向服务器的内容的，访问不同的路径出现不同的界面
• 查询字符串可以做到同一个页面不同内容
• 锚点是同一个内容不同位置，锚点不会上传到服务器
DNS 的作用是什么，nslookup 命令怎么用
DNS把域名和IP对应起来

IP 的作用是什么，ping 命令怎么用
IP用来访问目标服务器
域名是什么，分别哪几类域名
• com是公司的缩写
• org是非营利组织
• 有哪几类域名？

  1. com是顶级域名
  2. baidu.com是二级域名（俗称一级域名）
  3. www.baidu.com是三级域名（俗称二级）（其实是父子关系，但不是同一域名）
    例子：github.io把子域名xxx.github.io免费给大家使用
